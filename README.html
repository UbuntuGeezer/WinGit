1.0 Project Description.

The WinGit project supports Windows *cmd* files (.bat) on a Linux system. This allows developers to create "Windows-valid" .bat command files using Windows/DOS commands. These files, in turn, may be exported to a Windows-based system and execute properly with little or no modification.
